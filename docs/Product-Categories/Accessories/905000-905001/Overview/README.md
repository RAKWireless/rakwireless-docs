# 858~878MHz 8dBi Fiber Glass Antenna

## Features

-   Frequency Range: 858~878MHz
-   Max Gain: 8.0dBi
-   VSWR: ≤ 1.5
-   High efficiency
-   Vertically polarized dipole
-   Easy to mount with the included Installation kit
