# 860-930MHz 5dBi LoRaWAN Antenna Black Datasheet

### Features

- Frequency Range: **860~930&nbsp;MHz**
- Max Gain: **5.9&nbsp;dBi**
- VSWR: **≤ 2**
- High Efficiency
- Vertically Polarized Monopole
- N-type connector
- Easy to mount with the included Installation kit
